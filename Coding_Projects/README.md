# Coding Projects

This directory contains a collection of engineering-related coding projects
developed as part of coursework and independent study.

The projects span topics in aerospace, mechanical engineering, numerical
methods, and applied physics, with an emphasis on simulation, modeling,
and data analysis.

## Organization

Each project is contained within its own subdirectory and includes:
- Source code or notebooks
- Input data files (if applicable)
- A brief project-specific README describing the objective and methods

## Topics Covered

Example areas include (but are not limited to):
- Numerical methods and Monte Carlo simulation
- Rocket flight and trajectory modeling
- Data analysis and visualization
- Engineering computation and modeling

## Tools & Languages

Projects in this directory may use:
- Python (NumPy, SciPy, Matplotlib)
- Jupyter Notebooks
- CSV and other data formats
- Engineering and scientific computing workflows
